## 🧪 Tests for `RandomCandidateSelector`

This test suite verifies the correctness, robustness, and expected side effects of the `select_candidate` method. The method operates on an RDF graph and is responsible for selecting a valid candidate at random and annotating it as selected.

---

### 📌 Overview of behavior under test

The `select_candidate` method:
* Tests full functionality of `select_candidate` method.

* Scans an RDF graph for resources of type `slowmo:Candidate`

* Filters candidates to include only those that have at least one `slowmo:AcceptableBy` relationship

* Randomly selects one of the valid candidates

* Adds a triple to the graph marking the selected candidate:

  `slowmo:Selected true`

* Returns the selected candidate as an `rdflib.resource.Resource`
